Purpose

The X-CUBE-AZURE-TELEMATICS (STM32 Azure IoT software for STM32Cube) provides ready-to-run firmware examples to support quick evaluation and development of Azure IoT Cloud applications on B-U585I-IOT02A. Please refer to X-CUBE-AZURE v2 package for a secured implementation with TFM Trustzone security and STSAFE device.

  • IAR EWARM IDE and STM32CubeIDE are supported

  • X-CUBE-AZURE-TELEMATICS runs on following board:

    • B-U585I-IOT02A supporting Wi-Fi™ connectivity with an on-board EM3080 module.

  • It contains BSP compatible sensor drivers for B-U585I-IOT02A.

  • It connects to Azure IoT hubs through an MQTT over TLS connection.

  • It runs with Azure RTOS and NetXDuo network protocol stack.

  • One example application is provided, with STM32CubeMX .ioc files.

  • List of references to user documents:

    • Nx_Azure_IoT application

Known Limitations

  • if the the WiFi SSID name or password entered in configuration is wrong, the the application will block forever at “Get IP address…” step. Please check the SSID name and password, compile and flash with the correct configuration.

Development Toolchains and Compilers

  • For Nx_Azure_IoT application:
    • STM32CubeIDE v1.9.0
    • IAR Embedded Workbench for ARM (EWARM) toolchain: it is possible to start from IAR EWARM version V8.50.9 with patches for STM32U575/585 devices available here:
      Utilities\PC_Software\IDEs_Patches\EWARMv8_STM32U57x-58x_V1.3.zip
